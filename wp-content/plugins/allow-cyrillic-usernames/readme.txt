Allows users to register with Cyrillic usernames.

== Description ==

Allows to create users with usernames containing Cyrillic characters.

== Installation ==

1. Upload `allow-cyrillic-usernames` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
